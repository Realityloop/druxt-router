# Getting started

DruxtJS requires a Nuxt.js frontend and a Drupal JSON:API backend:

# Quickstart

  1. Install the NPM module:

    npm i druxt-router
    1

  2. Add the module and configure as required in nuxt.config.js:

    module.exports = {
  modules: [
    'druxt-router',
  ],
  druxt: {
    baseUrl: 'https://demo-api.druxtjs.org'
  }
}
