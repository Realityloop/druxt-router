# DruxtRouterNuxtModule : function

The Nuxt.js module function.

Installs the module functionality in a Nuxt application.

See: DruxtRouterNuxtModule
Example (nuxt.config.js)

module.exports = {
  modules: [
    'druxt-router'
  ],
  druxt: {
    baseUrl: 'https://demo-api.druxtjs.org'
  }
}
